Setup

Required Materials

This version is meant to be played in-person, not online; so grab a couple of friends!
You'll want three devices to play. One for each Spymaster and one to function as the public board.
Typically, the Spymasters will use smartphones and you'll use something larger to be the public board.
Navigate to this webpage on all three devices.

Seed Synchronization

You'll want to make sure that everyone is playing the same game.
Check the value of the Seed below, and make sure it's the same on all three devices.
If they aren't, everyone should agree on a Seed to use and type it into the Seed box below and hit Enter or the Set Seed Button

How to Play

Some things are a little different in this digital version.

For the public board...
Click the 'No!' button below, as you are not a Spymaster.
Any tile that you click on will cycle through all possible values: Red, Blue, Civilian, and Assassin.
After choosing a tile, you'll have to ask your Spymaster if you were right.

For Spymasters...
Click the 'Yes!' button below.
The true colors should be revealed to you now. Make sure to hide this from other operatives.
You can click on tiles to hide their text.
This can reduce visual clutter and indicate which tiles have already been chosen.
